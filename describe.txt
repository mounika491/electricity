10. generate_bill_task1.php:
1. INPUT:
   - Unique service number validation
   - Alphabets-only name validation
   - 10-digit phone validation
2. VALIDATION:
   - Reject duplicate service numbers
   - Reject invalid names/phones
3. COMPUTATION:
   - Slab rates: 1.5, 2.5, 3.5, 4.5
   - Minimum charge: ₹25 for 0 units
   - Fine: ₹150 for previous dues
4. OUTPUT:
   - Complete bill display
   - Due date with/without fine
FEATURES:
- Real-time validation
- Rate table display
- Requirements summary
